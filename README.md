# Hotel Management System (HMS) - Software Engineering Project

## 📝 Project Overview
This project focuses on building a Hotel Management System (HMS) to optimize the room booking process, guest information management, and billing. The application is developed as a Desktop Application with a modern user interface and a reliable relational database.

## 🛠 Technologies Used
* **Programming Language:** Python 3.x
* **GUI Framework:** CustomTkinter (Modern UI for Python)
* **Database:** SQLite (Lightweight relational database)
* **Project Management:** Jira (Kanban Board)
* **Design Tools:** Figma (UI/UX), app.diagrams.net (UML Diagrams)
* **Development Environment:** VS Code, Virtual Environment (venv)
